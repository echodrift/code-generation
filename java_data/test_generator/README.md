# Requirement
- apache-maven 3.6.3
- java 17
- javac 17
- randoop 4.3.3
- Jar files:
    - hamcrest-core-1.3.jar
    - junit-4.12.jar

# Config
- Store jar files and randoop 4.3.3 in lib folder
- Make sure that all dependencies of project is in target/dependency folder, using: ```mvn dependency:copy-dependencies```
